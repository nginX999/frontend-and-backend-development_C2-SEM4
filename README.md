Комплексное практическое задание по курсу "Фронтенд и бэкенд разработка"
Общее описание проекта
Данный репозиторий содержит выполнение комплекса практических занятий по курсу "Фронтенд и бэкенд разработка" (4 семестр, 2025/2026 уч. год). В рамках работы было создано полноценное веб-приложение интернет-магазина "HomeGoods" (товары для дома), объединяющее все изученные технологии и подходы.

Проект демонстрирует навыки работы с:

CSS-препроцессорами (SASS)

Серверной разработкой на Node.js + Express

Работой с JSON и тестированием API через Postman

Интеграцией фронтенда (React) с бэкендом

Документированием REST API с помощью Swagger

Практическое занятие №1: CSS-препроцессоры
Реализовано:
Использование SASS (SCSS) для стилизации приложения

Применение переменных для цветовой схемы и основных значений

Создание миксинов для переиспользуемых стилей (карточки, контейнеры)

Вложенная структура селекторов, повторяющая HTML-иерархию

Модульная организация стилей

Примеры использования:
scss
// Переменные
$bg: #0a0c14;
$primary: #6366f1;
$border: rgba(255, 255, 255, 0.08);

// Миксин для карточки
@mixin card($radius: 12px) {
  border-radius: $radius;
  border: 1px solid $border;
  background: rgba(255, 255, 255, 0.03);
}

// Вложенность
.product-card {
  @include card(20px);
  
  &__title {
    font-size: 20px;
  }
  
  &:hover {
    transform: translateY(-2px);
  }
}
Практическое занятие №2: Сервер на Node.js + Express
Реализовано:
Полноценное REST API для управления товарами интернет-магазина

CRUD операции (Create, Read, Update, Delete)

Middleware для логирования запросов

Валидация входящих данных

Обработка ошибок (404, 500)

Хранение данных в JSON-файле (имитация базы данных)

Маршруты API:
Метод	Путь	Описание
GET	/api/products	Получение списка всех товаров
GET	/api/products/:id	Получение товара по ID
POST	/api/products	Создание нового товара
PATCH	/api/products/:id	Обновление товара
DELETE	/api/products/:id	Удаление товара
Структура товара:
javascript
{
  id: "kDm3nP",           // Уникальный ID (nanoid)
  name: "Сковорода",       // Название
  category: "Посуда",      // Категория
  description: "...",      // Описание
  price: 2499,             // Цена
  stock: 15,               // Количество на складе
  rating: 4.5,             // Рейтинг (опционально)
  image: "url"             // Фото (опционально)
}

Доступ к документации:

text

Пример аннотации:
javascript
/**
 * @swagger
 * /api/products:
 *   get:
 *     summary: Возвращает список всех товаров
 *     tags: [Products]
 *     responses:
 *       200:
 *         description: Список товаров
 *         content:
 *           application/json:
 *             schema:
 *               type: array
 *               items:
 *                 $ref: '#/components/schemas/Product'
 */

Инструкция по запуску
Предварительные требования:
Установленный Node.js (v14 или выше)

Установленный npm

Запуск бэкенда:
bash
cd backend
npm install
npm start
# Сервер запустится на http://localhost:3000
# Swagger UI доступен на http://localhost:3000/api-docs
Запуск фронтенда:
bash
cd frontend
npm install
npm run dev
# Приложение запустится на http://localhost:3001
