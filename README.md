# Интернет-магазин - Fullstack приложение с аутентификацией и RBAC
---
## 📌 Описание проекта

Полноценное веб-приложение интернет-магазина с системой аутентификации и управления доступом на основе ролей (RBAC). Проект включает:

- **Бэкенд** на Node.js/Express с JWT аутентификацией
- **Фронтенд** на React с управлением товарами и пользователями
- **Систему ролей**: Администратор, Продавец, Пользователь, Гость
- **Документацию API** через Swagger

---

## Технологии

### Бэкенд
| Технология | Назначение |
|------------|------------|
| Node.js | Среда выполнения |
| Express.js | Веб-фреймворк |
| bcrypt | Хеширование паролей |
| jsonwebtoken | JWT токены (access + refresh) |
| nanoid | Генерация уникальных ID |
| cors | CORS настройки |
| swagger-jsdoc | Генерация Swagger документации |
| swagger-ui-express | UI для Swagger |

### Фронтенд
| Технология | Назначение |
|------------|------------|
| React.js | Библиотека для UI |
| React Router | Маршрутизация |
| Axios | HTTP запросы с interceptors |
| Context API | Управление состоянием (аутентификация) |

## 📋 API Маршруты

### Аутентификация

| Метод | Маршрут | Описание | Доступ |
|-------|---------|----------|--------|
| POST | `/api/auth/register` | Регистрация нового пользователя | 🔓 Гость |
| POST | `/api/auth/login` | Вход в систему (выдаёт токены) | 🔓 Гость |
| POST | `/api/auth/refresh` | Обновление пары токенов | 🔓 Гость |
| POST | `/api/auth/logout` | Выход из системы | 🔒 Пользователь+ |
| GET | `/api/auth/me` | Получение текущего пользователя | 🔒 Пользователь+ |


### Пользователи (только Администратор)

| Метод | Маршрут | Описание | Доступ |
|-------|---------|----------|--------|
| GET | `/api/users` | Список всех пользователей | 🔒 Администратор |
| GET | `/api/users/:id` | Получение пользователя по ID | 🔒 Администратор |
| PUT | `/api/users/:id` | Обновление информации пользователя | 🔒 Администратор |
| DELETE | `/api/users/:id` | Блокировка пользователя | 🔒 Администратор |


### Товары

| Метод | Маршрут | Описание | Доступ |
|-------|---------|----------|--------|
| GET | `/api/products` | Список всех товаров | 🔒 Пользователь+ |
| GET | `/api/products/:id` | Получение товара по ID | 🔒 Пользователь+ |
| POST | `/api/products` | Создание нового товара | 🔒 Продавец+ |
| PUT | `/api/products/:id` | Обновление товара | 🔒 Продавец+ |
| DELETE | `/api/products/:id` | Удаление товара | 🔒 Администратор |

> 🔓 - открытый доступ | 🔒 - требует аутентификацию


## 🎯 Система ролей (RBAC)

| Роль | Описание | Права доступа |
|------|----------|---------------|
| 👤 **Гость** | Не аутентифицированный пользователь | Регистрация, вход |
| 👤 **Пользователь** | Обычный пользователь сайта | Просмотр товаров, просмотр профиля |
| 🛒 **Продавец** | Сотрудник сайта | Просмотр + создание + редактирование товаров |
| 👑 **Администратор** | Управленец сайта | Все права продавца + управление пользователями |

## 🔐 Безопасность

| Механизм | Описание |
|----------|----------|
| **Хеширование паролей** | bcrypt с солью (10 раундов) |
| **JWT токены** | Access токен (15 мин) + Refresh токен (7 дней) |
| **Ротация токенов** | При обновлении старый refresh токен удаляется |
| **Защита маршрутов** | Middleware для проверки токена и ролей |
| **CORS** | Разрешены только запросы с фронтенда (http://localhost:3001) |
